# Voice Assistant

This is a simple voice assistant developed using Python. The project recognizes voice commands and performs basic tasks such as opening websites, launching applications, searching Google, and telling the current time and date.

## Features

- Voice command recognition
- Open websites (YouTube, Google, GitHub, etc.)
- Open Windows applications (Calculator, Notepad, Paint)
- Search anything on Google
- Tell the current time and date
- Open Gmail
- Simple text-based assistant
- Basic browser interface using HTML

## Technologies Used

- Python
- SpeechRecognition
- PyAudio
- pyttsx3
- HTML
- JavaScript

## Project Files

- `universal_assistant.py` - Main voice assistant
- `test_mic.py` - Test microphone
- `text_opener.py` - Text command assistant
- `app_opener.py` - Open applications and websites
- `assistant.html` - Simple browser interface

## How to Run

1. Install Python.
2. Install the required libraries.

```bash
pip install SpeechRecognition pyttsx3 requests pyaudio
```

3. Run the project.

```bash
python universal_assistant.py
```
